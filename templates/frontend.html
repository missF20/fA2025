Dana AI Frontend Access

Instructions to run the React frontend of Dana AI

About the Dana AI Frontend

The Dana AI Frontend is a full-featured React application built with:

  • React 18 with TypeScript
  • React Query for data fetching
  • Supabase for authentication and data storage
  • Framer Motion for animations
  • Tailwind CSS for styling

The frontend provides a complete user interface for interacting with the Dana AI Platform, including:

  • User authentication (login/signup)
  • Dashboard with metrics and visualizations
  • Conversation management
  • Knowledge base interface
  • Integration management
  • Subscription management

Running the Frontend Application

Navigate to the frontend directory
cd frontend
Install dependencies
npm install
Start the development server
npm run dev

This will start the React development server on port 5173 (default Vite port)

Access the frontend

The frontend will be available at: http://localhost:5173

For remote access, use: https://dana-ai-frontend.replit.app

Environment Configuration

The frontend requires the following environment variables:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_API_URL=http://localhost:5000/api
VITE_SENTRY_DSN=your_sentry_dsn (optional)
VITE_SENTRY_ENVIRONMENT=development

Create a .env.local file in the frontend directory with these variables.

Launch Dana AI Frontend

This will open in a new tab with the hosted React-based frontend application.
