Dana AI Platform Preview Instructions

✅ Services Running

The Dana AI Platform is running successfully on the following URLs:

  • Flask Backend: http://localhost:5000/
  • React Frontend: http://localhost:5174/

How to Access the Application

You can access the Dana AI Platform in two ways:

Option 1: Direct Access

Click the links below to open the application in your browser:

  • Open React Frontend
  • Open Flask Backend

Option 2: Replit Preview

If the direct links don't work, you can use the Replit preview feature:

  1. Look for the "Webview" tab in your Replit interface
  2. Click on it to open the application preview
  3. If the preview is blank, try manually entering http://localhost:5174/ in the preview URL bar

⚠️ Troubleshooting

If you're having issues accessing the preview:

  • Make sure the application is still running (check the console output)
  • Try restarting the workflow with restart_workflow
  • If using a mobile device, ensure you're on the same network as your development environment
  • Check browser console for any JavaScript errors

Token Usage Widget

To view the Token Usage Widget:

  1. Log in to the application
  2. Navigate to the Conversations page
  3. The Token Usage Widget should appear at the top right of the Conversations dashboard

The widget displays:

  • Current token usage with a color-coded progress bar
  • Remaining token count
  • A settings modal to adjust token limits for AI responses
